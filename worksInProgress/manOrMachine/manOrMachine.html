Man or Machine

This project explores what humans expect of interactions with robots. The project consists of two separate interaction sites: one is at the robot, the other is at a station where humans chose how the robot will behave. Ideally these two parts are sufficiently separated that users at each station don't know about, or at least can't see, the others.

The robot is slightly less than human height, so that people interact with it face to face. The robot wanders aimlessly avoiding obstacles and searching for humans, which it detects visually. Once the robot finds a human, it starts a conversation. The robot can speak and make various facial expressions.

The robot asks simple questions by speaking through a loudspeaker. Visitors answer these questions using buttons on the robot. Depending on the answers, the robot reacts in a certain way, and asks another question. In this way the robot and visitor engage in a simple conversation.

At the other station visitors choose from a limited set of reactions for each answer to each question. In this way, these visitors create the robot's "personality".
